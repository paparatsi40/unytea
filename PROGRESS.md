## ğŸ“ **Notes**

### Technical Decisions Made

1. **Next.js 14 (App Router)** - Modern, server components, streaming
2. **Prisma** - Type-safe ORM, great DX
3. **tRPC** - End-to-end type safety without API docs
4. **Clerk** - Best-in-class auth, saves months of work
5. **Livekit** - Open-source WebRTC, better than Zoom API
6. **Tailwind** - Rapid UI development, easy theming
7. **PostgreSQL** - Reliable, scalable, full-featured

### Design Decisions Made

1. **HSL color system** - Easy to theme, better than RGB
2. **Glass morphism** - Modern, premium feel
3. **Smooth animations** - Not too fast, not too slow (200-400ms)
4. **Mobile-first** - Responsive from day one
5. **Accessibility** - WCAG 2.1 AA compliance target
